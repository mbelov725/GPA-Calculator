# GPA Calculator
A simple command-line GPA calculator written in **C++**. 
This project allows users to input course information and calculate GPA based on course weights and percentages. The calculator follows **University of Waterloo GPA conversion and credit weight standards**.

## Features
- Add multiple courses
- Store course code, name, grade percentage, and weight
- Modify and delete courses
- Display a transcirpt
- Convert percentage grades to GPA values
- Calculate overall GPA
